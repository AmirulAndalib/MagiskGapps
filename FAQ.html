FAQ

  • What is MagiskGApps?

    MagiskGApps is a custom package of Google apps and services designed specifically for devices running Magisk, a popular root management tool. It allows you to install Google apps without modifying the system partition, keeping the device root intact while offering flexibility in which apps to install.

  • How does MagiskGApps work?

    MagiskGApps works by leveraging Magisk’s systemless root system. When you install MagiskGApps, it adds Google apps and services through Magisk’s recovery interface, ensuring that these apps do not modify the system partition. This method ensures that your device remains rooted and customizable while keeping the integrity of your system intact.

  • What apps are included in MagiskGApps?

    MagiskGApps comes with several Google apps like the Play Store, Gmail, Google Play Services, Google Maps, and others. The package is modular, so you can choose which apps you wish to install based on your preferences, helping you avoid unnecessary bloatware.

  • Can I install MagiskGApps on any rooted device?

    MagiskGApps is designed specifically for devices running Magisk. If your device is rooted with Magisk, you can install MagiskGApps. However, it may not be compatible with other root management tools or if the device is not rooted.

  • Is MagiskGApps safe to use?

    Yes, MagiskGApps is safe as long as you are using it on a compatible device with Magisk installed. It operates in a systemless environment, meaning it doesn’t alter the core system files, thus preventing the risk of bricking or corrupting the system partition.

  • Do I need to wipe my device to install MagiskGApps?

    No, MagiskGApps does not require a factory reset or wiping of data. It can be flashed directly through the Magisk Manager app or recovery without affecting your personal data. However, always make sure to back up your data just in case.

  • Can I uninstall MagiskGApps if I no longer need it?

    Yes, MagiskGApps can be uninstalled by simply flashing the Magisk zip again without the GApps package or by removing the apps manually using a file manager. However, it's recommended to use Magisk Manager for a clean removal process.

  • Will MagiskGApps work on custom ROMs?

    Yes, MagiskGApps is compatible with most custom ROMs, as long as they support Magisk. It works with AOSP-based, LineageOS, and many other custom Android ROMs.

  • Does MagiskGApps affect system updates?

    MagiskGApps does not interfere with system updates because it doesn’t modify the system partition. However, keep in mind that installing updates might require re-flashing MagiskGApps if the update affects the Magisk root or Google services.

  • Where can I download MagiskGApps?

    MagiskGApps can be downloaded from trusted sources like the official Magisk GitHub repository or reputable Android forums such as XDA. Be cautious of downloading from unofficial sites to avoid any potential security risks.

FAQ

If a Google app is updated after the module is installed, the app may stay on the device, even if the module is removed. The updated apps can be uninstalled, as they do not remain as system apps after the module is removed.

This is due to the original GApps packages being designed to show device setup on first boot after the flash. But when the MGM module is flashed, it still carrys over the device setup app. Disableing this app should fix this issue, and a perminant fix is in the works.
